# OpenAI just launched Operator - Its first AI agent

**Author:** Katie Hignett
**Date:** January 23, 2025
**Source:** https://www.indiehackers.com/post/mLMf972SGpK9XGh723pH

---

OpenAI has unveiled "Operator," a browser-based AI agent capable of automating everyday tasks like restaurant bookings and form completion. The agent interfaces with websites through a new computer-use model paired with GPT-4o's vision capabilities, allowing it to interpret graphical interfaces and execute instructions.

Currently in limited release for U.S.-based ChatGPT Pro subscribers, the tool will expand to Plus tier users in coming months, with European Union availability delayed due to regulatory considerations.

The underlying Computer Using Agent model demonstrates strong performance on web-based benchmarks (87% on WebVoyager, 58.1% on WebArena) but is less effective for full computer tasks (38.1% on OSWorld). This outperforms Anthropic's Claude, which achieved up to 22% on similar tests upon its October launch, though Claude maintains API availability advantages for developers.

OpenAI emphasizes safety through mandatory user confirmation protocols, with sensitivity-based escalation and prompt injection detection features. The company deployed gradual rollout strategies to manage risks associated with autonomous agent technology.

CEO Sam Altman characterized this as "really the beginning" of the product, with additional agents launching in subsequent weeks. The CUA model will become accessible via API shortly, enabling third-party developer integration.

## Key Lessons

- Domain-specific agents with strong security guarantees may outcompete general-purpose tools for sensitive operations
- Gradual deployment and user control mechanisms are essential for autonomous agent adoption
- Web-based task automation represents a practical near-term application before broader computer control

## Tech Stack Mentioned

- GPT-4o (vision capabilities)
- Computer Using Agent model (proprietary)
- Cloud-based browser environment
- Browser automation technology
