# All the best products from Anthropic's Builder Day hackathon

**Author:** Katie Hignett
**Date:** November 7, 2024
**Source:** https://www.indiehackers.com/post/GRrcvZG78DdFyn2V1JJs

---

Anthropic held its inaugural "Builder Day" hackathon at San Francisco offices with Menlo Ventures, attracting over 200 developers to build applications using Claude AI. The event featured talks from Anthropic CEO Dario Amodei and Head of Claude Relations Alex Albert on recent Claude updates.

Judges selected three winning products from approximately 100 entries. First place went to a robotic arm controller that used Claude's computer vision and reasoning capabilities. Developers uploaded an Amazon robot's instruction manual to Claude, enabling it to execute commands like "rotate left until you find a human" while processing live camera feeds.

Second place recognized an anti-captcha application described as a "honeypot for AI agents," using Claude to generate captchas solvable by humans but not AI systems.

Third place featured AI agents assuming different professional roles (UX lead, data scientist, finance manager, CEO) to debate and improve product requirements documents automatically.

**Key Lessons:**
- Claude's computer use capabilities enable real-world hardware control
- Context-based learning through documentation improves AI agent performance
- Multi-agent systems can replicate collaborative human workflows

**Tech Stack Mentioned:**
- Claude AI (large language model)
- Amazon robotic arm
- Claude Haiku model
- AgentOps AI platform
