# 8 vibe coders who are building real SaaS MVPs, not just games

**Author:** Channing Allen
**Date:** April 4, 2025
**Source:** https://www.indiehackers.com/post/ItN5XsfrbQ2RCwawoIqn

---

The piece showcases eight founders leveraging AI-assisted coding to rapidly develop functional SaaS products. The article emphasizes that "AI is writing 90-100% of the code," enabling developers to launch MVPs in days rather than months.

## Eight Featured Projects

1. **Analytics Tool** (Courtland Allen) - Visualizes website visitors as a city environment with real-time navigation tracking; 95% AI-generated code
2. **Voice-Navigated Recipe Site** (Tom Blomfield) - Voice-controlled recipe interface for hands-free cooking assistance; built in weeks
3. **SEO ROI Calculator** (Chris Tweten) - Estimates revenue potential and timeline from SEO investments
4. **Penetration Testing Tool** (Simo Rachidi) - Multi-agent AI security scanner; completed in two days
5. **Content Creation SaaS** (Sebastian Volkis) - Identifies viral trending articles and generates short-form content; £5.5k revenue on launch day
6. **Affiliate Marketing Platform** (Ollie Efez) - Streamlined affiliate recruitment and management system
7. **Mock API Tool** (Raj Savaliya) - Generates realistic test data for frontend development without backend delays
8. **AI-Powered Form Builder** (Amrish J./Chadform) - Adaptive forms that adjust questions based on respondent input

## Technology Stack Referenced

- **IDEs/Tools:** Cursor, Windsurf
- **AI Models:** Claude Sonnet 3.7, Claude Code, OpenAI's o1 pro mode, Gemini 2.5 Pro, OpenAI voice mode API
- **Additional Technologies:** React, TypeScript, PostgreSQL, Prisma ORM

## Revenue Data

- **Sebastian Volkis (Content Tool):** £5.5k first-day revenue post-launch

## Key Lessons

- Successful vibe coding requires foundational programming knowledge and debugging skills
- Understanding of project structure and code organization is essential
- Ability to handle edge cases and security considerations matters
- Testing and validation practices for scaling are important
- Proper database modeling expertise is needed
- "Every time I vibe code, I learn new things" — the approach accelerates skill development alongside MVP creation
