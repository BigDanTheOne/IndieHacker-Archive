# Claude's new model 4x more expensive, Mistral launches a moderation API

**Author:** Darko Gjorgjievski
**Date:** November 9, 2024
**Source:** https://www.indiehackers.com/post/aZlB7gS03Mts405zgLWR

---

The piece covers two significant AI developments:

1. Anthropic increased pricing for its Haiku model -- the new version costs substantially more than before, creating potential cost concerns for developers integrating Claude's models.

2. Mistral unveiled a content moderation API powered by its AI models, supporting multiple languages for global applications.

**Key Lessons:**
- Premium pricing for advanced AI features is becoming standard industry practice
- Multilingual support in moderation tools addresses global market needs
- Cost management matters for developers relying on third-party AI services

**Tech Stack Mentioned:**
- Claude (Anthropic's AI model)
- Haiku (Claude model variant)
- Mistral AI models
- Content moderation API
