# Google accidentally leaks its web-browsing AI agent

**Author:** Darko Gjorgjievski
**Date:** November 7, 2024
**Source:** https://www.indiehackers.com/post/rILxynWHmFfeTQfstJ2j

---

Google accidentally leaks its web-browsing AI agent. Its codename is "Jarvis," and it was described as a "helpful companion that surfs the web with you." Jarvis will supposedly allow you to automate your web browsing tasks from within Chrome, such as shopping or travel. Claude already has an AI agent in beta.

**Key Lessons:**
- Google developed an unreleased web-browsing AI agent codenamed Jarvis
- The tool aims to automate routine web tasks through Chrome integration
- Competitor Claude has already launched an AI agent in beta testing

**Tech Stack Mentioned:**
- Chrome (browser integration platform)
- Claude (competing AI agent platform)
