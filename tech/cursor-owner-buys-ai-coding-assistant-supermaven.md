# Cursor owner buys AI coding assistant Supermaven

**Author:** Katie Hignett
**Date:** November 13, 2024
**Source:** https://www.indiehackers.com/post/UJu7H1MTEfYx0Yy871jV

---

Anysphere, the company behind AI code editor Cursor, has acquired Supermaven, an AI coding assistant. The acquisition will enhance Cursor's Tab autocomplete feature, making it "context aware" and better equipped to handle extended code sequences.

Supermaven launched nine months prior and introduced features including in-editor chat, code deletion support, and codebase navigation capabilities. The team also significantly upgraded their Babble AI model.

Jacob Jackson, Supermaven's CEO and creator of Tabnine (the first AI code completion tool in 2019), noted: "the next step for the product was not just smarter models, but models with new capabilities." He previously worked at OpenAI.

Supermaven will maintain existing plugins for VSCode, JetBrains, and Neovim while the primary focus shifts to Cursor. The acquisition price remains undisclosed.

**Key Lessons:**
- API constraints limit product innovation in plugin-based architectures
- Acquiring complementary technology accelerates feature development
- Building custom infrastructure enables capabilities impossible within external frameworks

**Tech Stack Mentioned:** VSCode, JetBrains, Neovim, Tabnine, ChatGPT
