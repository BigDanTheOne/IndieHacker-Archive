# 8 vibe coders who are building real SaaS MVPs, not just games

**Author:** Channing Allen
**Date:** April 4, 2025
**Source:** https://www.indiehackers.com/post/tech/8-vibe-coders-who-are-building-real-saas-mvps-not-just-games-ItN5XsfrbQ2RCwawoIqn

The piece highlights founders leveraging AI-assisted coding to rapidly prototype functional SaaS products. According to the article, "Founders are using AI to build real SaaS MVPs in days, not months. AI is writing 90-100% of the code."

## Featured Projects

1. **Analytics Tool** - Courtland Allen created a visualization tool transforming website analytics into an explorable city interface, with 95% AI-generated code using Cursor and Claude.

2. **Voice-Controlled Recipe Site** - Tom Blomfield built a voice-navigable cooking application enabling hands-free recipe guidance.

3. **SEO ROI Calculator** - Chris Tweten developed estimation software projecting revenue potential and timelines for search optimization efforts.

4. **Security Testing Tool** - Simo Rachidi created an open-source penetration testing application completed within two days.

5. **Content Generation Platform** - Sebastian Volkis built a trend-monitoring system generating short-form content, achieving 5.5k GBP in first-day revenue.

6. **Affiliate Marketing Platform** - Ollie Efez developed a streamlined affiliate recruitment and management system.

7. **Mock API Generator** - Raj Savaliya created development infrastructure enabling frontend work without backend dependencies.

8. **Adaptive Form Builder** - Amrish J. developed Chadform, an AI-responsive questionnaire system adjusting inquiries based on user responses.
